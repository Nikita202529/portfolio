Правки стилей лучше вносить в файл scss, после чего конвертировать в min.css с помощью любого компилятора scss (например, раширение Live Sass Compiler от Ritwick Dey для программы Visual Studio Code).
Второй вариант, похуже - создать свой отдельный css файл с правками и добавить его в тег <head> файла с html страницы с помощью еще одного тега <link>.
